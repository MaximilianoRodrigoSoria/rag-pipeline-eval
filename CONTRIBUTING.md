# Contribuir

Gracias por tu interés en mejorar este proyecto.

## Flujo
1. Hacé un fork y creá una rama: `feature/mi-cambio` o `fix/mi-arreglo`.
2. Hacé tus cambios, enfocados (un tema por PR).
3. Corré los tests: `python -m pytest -q` (con `PYTHONPATH=src` si aplica).
4. Usá [Conventional Commits](https://www.conventionalcommits.org/) (`feat:`, `fix:`, `docs:`, `chore:`).
5. Abrí un Pull Request describiendo qué cambia y por qué.

## Estilo
- Código en inglés donde sea idiomático; documentación en español.
- Mantené el código mínimo y legible.
- Actualizá la documentación si tu cambio lo amerita.
